About

This app is primary intended for students of the MECS study program, though anyone can use it to practise their Mandarin.

You can enter characters using Pinyin or directly if you have a Chinese keyboard/input method enabled. If you get stuck you can use the small hint button above the input box to find out what the next character should be.

Read the blog post and get the code from github.
